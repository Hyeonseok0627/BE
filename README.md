### 2. 주요 기능

- Spring Security, JWT를 이용한 회원가입/로그인
- 이메일 인증을 통한 로그인 구현
- AWS S3를 이용한 다중 이미지 업로드
- JPA Pageable을 이용한 페이지 무한 스크롤
- 타입별, 가격별 필터링 기능
- 키워드 검색 기능
- 숙소 좋아요 기능
- swagger 적용
- 다중 이미지 업로드 CRUD(조회 시 이미지 preview)

### 4. 기술 스택
 
#### 🎨 **Front-end Stack**

- React , javascript
- Redux
- Redux Toolkit
- mui , styled-components
- axios

#### 🛠 **Back-end Stack**

- Spring Security, JWT
- AWS S3, RDS(MySQL)
